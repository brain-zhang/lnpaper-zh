## 术语翻译

* Preimage / Input to hash: 哈希原像
* Near-infinite scalability: 近乎无限的可扩展性
* Malleability: 延展性
* Soft-fork: 软分叉
* Micro-payment channel: 微支付通道
* Funding Transaction：筹资交易 / 保证金交易
* Commitment Transaction: 承诺交易
* Delivery Transaction: 交付交易 / 传送交易
* Revocable Delivery Transaction: 可撤销交付交易
* Breach Remedy Transaction: 违约补救交易
* Hashed Timelock Contract (HTLC): 哈希时间锁合约
* Revocable Sequence Maturity Contract (RSMC): 可撤销序列成熟度合约
* Net Settlement: 净额结算
* Trustless: 去信任化 / 无需信任
* Timestop: 停止计时
* Output: 交易输出
* Input: 交易输入
